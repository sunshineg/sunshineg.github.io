# Sunny Gao (高阳) - Personal Website

🌐 **Live Website:** [sunshineg.github.io](https://sunshineg.github.io)

This repository contains the source code for Sunny Gao's personal portfolio and bio website, featuring a "geeky" terminal-style aesthetic and cyberpunk design elements.

## About Sunny

Sunny is a distinguished young entrepreneur and builder. He serves as the Co-founder & ex-CEO of the developer community SegmentFault and is the initiator of the open-source software Apache Answer.

In an era where AI is fundamentally transforming how people acquire and learn information, as well as reshaping the paradigms of research and creation, Sunny embraces this rapid evolution. As an expert in community building, he firmly believes that open source will win the future. He is dedicated to advancing human technology and civilization by harnessing the collective wisdom of human-AI collaboration and promoting open knowledge sharing.

## Tech Stack & Features

This website is designed to be lightweight, fast, and visually striking without relying on heavy frameworks:

- **Core:** Pure HTML5, CSS3, and Vanilla JavaScript.
- **Visuals:** Implements a custom Matrix digital rain background, CSS glitch text effects (`@keyframes`), and a retro terminal typing animation.
- **Typography:** Fully optimized for mixed typography, ensuring half-width spacing rules between Chinese characters, English words, and numbers.
- **Deployment:** Automatically deployed and hosted via [GitHub Pages](https://pages.github.com/).
