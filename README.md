# Sunny Gao (高阳) - Personal Website

🌐 **Live Website:** [sunshineg.github.io](https://sunshineg.github.io)

This repository contains the source code for Sunny Gao's personal portfolio and bio website, featuring a "geeky" terminal-style aesthetic and cyberpunk design elements.

## Tech Stack & Features

This website is designed to be lightweight, fast, and visually striking without relying on heavy frameworks:

- **Core:** Pure HTML5, CSS3, and Vanilla JavaScript.
- **Visuals:** Implements a custom Matrix digital rain background, CSS glitch text effects (`@keyframes`), and a retro terminal typing animation.
- **Typography:** Fully optimized for CJK (Chinese-Japanese-Korean) mixed typography, ensuring half-width spacing rules between Chinese characters, English words, and numbers.
- **Deployment:** Automatically deployed and hosted via [GitHub Pages](https://pages.github.com/).
